# User Guide: 3Y Challenge Framework (NotebookLM)

## How to Use

1. Upload prospect research (discovery notes, tech stack, MEDDPICC data, website)
2. Paste the prompt and run
3. NotebookLM generates a 3-column challenge framework for your "Securing Prospects Growth" slide

---

## What You'll Get

3 columns, each containing:
- **Business Theme/Goal** (e.g., "Democratizing Security" or "Continuous Compliance")
- **3 Challenge Boxes** - Each with a header + 2-3 bullet points on business implications

Challenges are grounded in:
- Their actual tech stack (AWS/Azure/GCP, current tools)
- MEDDPICC pain points with quantifiable impact
- Prospect's own language from discovery

---

## Output Use

Use this to:
- Build executive presentation slides
- Connect technical gaps to business outcomes
- Structure discovery findings visually
- Align Wiz solutions to their strategic goals

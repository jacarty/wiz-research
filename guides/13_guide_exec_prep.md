# Executive Meeting Preparation Prompt

## Notes
This prompt helps you research and prepare for meetings with senior executives at UK companies. The output is optimized for upload to NotebookLM, which can then generate an audio briefing podcast.

**Tips for Best Results:**
- Be specific with executive names and titles
- Specify the industry/vertical for targeted insights
- Emphasize recent news (last 3-6 months)
- Request UK-specific regulatory and market context
- Always ask for source links

**Example:**
I'm meeting with **Jane Smith, CISO** at **Marks & Spencer** in the UK. Create a briefing document covering: [insert sections above]

## How to Use

1. Upload prospect research (contacts, discovery notes, tech stack, org chart)
2. Paste the prompt and run
3. NotebookLM generates an output file the key persona

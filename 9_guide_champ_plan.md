# User Guide: Champion Development Plan (NotebookLM)

## How to Use

1. Upload prospect research (contacts, discovery notes, tech stack, org chart)
2. Paste the prompt and run
3. NotebookLM generates a champion-building strategy for each key persona

---

## What You'll Get

A prioritized table with one row per contact/persona containing:

- **WHO & ROLE** - Title, influence level, decision authority
- **WHAT THEY CARE ABOUT** - Responsibilities, metrics, challenges
- **TYPICAL KPIs** - How Wiz impacts their performance scorecard
- **CHAMPION BUILDING IDEAS** - Value hooks, quick wins, enablement tactics, relationship strategies

Personas are ranked by influence and engagement readiness.

---

## Output Use

Use this to:
- Tailor messaging to each stakeholder's priorities
- Identify quick wins to build credibility
- Equip champions with tools to advocate internally
- Map multi-threaded engagement strategy
